<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-3">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get HiveMQ CE running in minutes with your preferred deployment method.</p>
  
  <Steps>
    <Step title="Download HiveMQ CE">
      Download the latest binary package or pull the Docker image.
      
      ```bash Docker
      docker pull hivemq/hivemq-ce
      ```
      
      Or download the binary package from [GitHub Releases](https://github.com/hivemq/hivemq-community-edition/releases).
    </Step>
    
    <Step title="Start the broker">
      Launch HiveMQ CE using Docker or the included run scripts.
      
      ```bash Docker
      docker run -d --name hivemq-ce -p 1883:1883 hivemq/hivemq-ce
      ```
      
      ```bash Binary
      cd hivemq-ce-2025.5
      bin/run.sh
      ```
      
      <Note>Java 11 or higher is required to run HiveMQ CE.</Note>
    </Step>
    
    <Step title="Connect your first client">
      Connect an MQTT client to `localhost:1883` and start publishing messages.
      
      ```python Python with paho-mqtt
      import paho.mqtt.client as mqtt
      
      client = mqtt.Client()
      client.connect("localhost", 1883, 60)
      client.publish("test/topic", "Hello from HiveMQ!")
      client.disconnect()
      ```
      
      Your MQTT broker is now ready to handle messages!
    </Step>
  </Steps>
</div>
